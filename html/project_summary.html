Project Summary

Get to Know this Xu Xin Project

This website is dedicated to Xu Xin, a world-renowned Chinese table tennis player known for his creativity, unique playing style, and professional achievements.

Quick Overview

The site aims to present a comprehensive yet accessible showcase of Xu Xin’s:

  • Biography and playing style
  • Major career achievements
  • Highlights and memorable moments
  • Personal brand “Xuperman” and associated products

The purpose of this site is to introduce to both fans and new people about Xu Xin’s contributions to the sport, in a balanced way of text, visuals, and multimedia content. It is designed as a mini-biography, product exhibition, and showing his influence in table tennis matches/cultures.

Contents

This website has 6 main pages

  • index.html - A homepage for everything
  • basic-info.html - An intro to Xu Xin's life and his playing style
  • achievements.html - A timeline & list of Xu Xin's participation and ranks in different matches
  • highlights.html - An exhibition of different clips picked for his best moments
  • xuperman.html - A page designed to show Xu Xin's business and talks about it in details
  • equipment.html - A page designed to exhibit commodities in Xuperman

Implemented Features

To make better vision effects and make everything more readable. This website Implemented different CSS and jQuery for better experiences.

  • External CSS for shared styling that linked across all pages: style.css
  • Internal CSS for page-specific styles: Each content page includes internal style blocks for elements unique to that page
  • Unified navigation menu and footer on all pages for consistent design
  • Responsive elements using flexbox
  • iframe embeds YouTube highlight videos on highlights.html
  • CSS transitions and transformations:
    • transform: scale() on hover for blocks, product cards, and images
    • transition: opacity for different vision effects on blocks on index.html
    • smooth transition effects for interactive elements
